# useCustomCompareMemo

Like `useMemo` but with a callback to defines that factory function can be invoked.

  <Story story={Example} inline />

## Reference

```ts
type DepsAreEqual<TDeps extends DependencyList> = (savedDeps: TDeps, deps: TDeps) => boolean;

function useCustomCompareMemo<TFactory extends () => unknown, TDeps extends DependencyList>(
  factory: TFactory,
  deps: TDeps,
  depsAreEqual: DepsAreEqual<TDeps>
): ReturnType<TFactory>;
```

#### Importing

<ImportPath />

#### Arguments

- **factory** _`() => unknown`_ - useMemo factory function.
- **deps** _`DependencyList`_ - useMemo dependency list.
- **depsAreEqual** _`DepsAreEqual<DependencyList>`_ - defines that factory function can be invoked.
